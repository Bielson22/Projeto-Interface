1. Interface Home (Página Inicial)
Projetada para ser o ponto de partida imersivo e direto para o usuário.

Navegação (RF01): Possui uma barra superior (Navbar) fixa com efeito visual moderno, organizando o acesso rápido às seções de Home, Dicas, Denúncias e Calculadora.

Hero Section: Apresenta uma mensagem de impacto ambiental com o botão "Começar Agora", que incentiva o início imediato da experiência.

Bento Grid: Um conjunto de três cards que separa as funções do site de forma visual e intuitiva.

2. Interface de Dicas Sustentáveis
Focada no pilar educativo do projeto para transformar consciência em ação.

Conteúdo Informativo (RF02): Disponibiliza dados sobre economia de água e gestão de resíduos para qualquer visitante.

Objetivo: Oferecer soluções práticas após o usuário descobrir seu impacto no monitoramento.

3. Interface do Canal de Denúncias
Espaço dedicado ao engajamento coletivo e fiscalização ambiental.

Formulário Externo (RF05): Para garantir a organização e segurança dos dados, o registro de denúncias (como queimadas ou desmatamento) é feito via Google Forms integrado ao fluxo do site.

Ação Individual: Permite que o cidadão reporte incidentes de forma simples e direta.

4. Monitor Atmosférico (Calculadora de CO₂)
A ferramenta central de interatividade e processamento de dados do sistema.

Funcionamento (RF04): O usuário ajusta barras deslizantes de transporte e energia para calcular sua emissão mensal de carbono.

Feedback Visual: O sistema exibe o resultado em kg de CO₂ e uma barra de "Pureza do Ar" que muda de cor (Verde para "Ótima", Vermelho para "Crítica").

Acesso Restrito (RF02/RF03): Embora os dados informativos sejam públicos, o sistema exige Login ou Cadastro para processar cálculos profundos e salvar o histórico de resultados no perfil do usuário.
